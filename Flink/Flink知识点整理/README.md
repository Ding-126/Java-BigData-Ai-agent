# Flink 面试点整理（60k 岗位面试官视角）

本目录为 **Flink 面试点整理**，站在约 **60k 岗位面试官** 视角系统提出面试问题并整理详细答案，供候选人复习与面试官备课使用。

## 文件列表与题目统计

| 文件 | 模块（中文） | 题目数 | 说明 |
|------|--------------|--------|------|
| [01-architecture.md](./01-architecture.md) | 架构与基础 | 14 题 | 涵盖架构设计、调度算法、高可用、大规模平台设计 |
| [02-api.md](./02-api.md) | API 与编程模型 | 17 题 | 涵盖 API 原理、窗口、Watermark、CEP、源码实现 |
| [03-state-fault-tolerance.md](./03-state-fault-tolerance.md) | 状态与容错 | 17 题 | 涵盖状态管理、Checkpoint、容错机制、状态迁移 |
| [04-performance.md](./04-performance.md) | 性能与调优 | 15 题 | 涵盖背压、数据倾斜、GC 优化、大规模性能优化 |
| [05-production.md](./05-production.md) | 生产与运维 | 18 题 | 涵盖监控告警、故障排查、平台设计、A/B 测试 |

**总计**：**81 道面试题**，每题均配有详细答案。

## 难度分级

- **[基础]**：适合 30-40k（概念、基本用法、配置）
- **[进阶]**：适合 40-50k（原理、调优、常见问题）
- **[高级]**：适合 50-60k（源码理解、架构设计、复杂场景）
- **[专家]**：适合 60k+（深度优化、故障排查、平台设计）

## 推荐阅读顺序

- **按模块顺序（系统复习）**：01 → 02 → 03 → 04 → 05。
- **按难度（分层刷题）**：先读各文件中标为 [基础] 的题，再 [进阶]，最后 [高级]/[专家]；跨文件可按「基础 → 进阶 → 高级 → 专家」路径跳转。

## 版本与约定

- **答案以 Apache Flink 1.17 为主**；若某题涉及版本差异，会在答案中单独标明「适用版本」或「1.17 与 1.13 差异」。
- **答案结构**：每个答案包含「是什么」「为什么这样设计」「如何实现（源码思路）」「如何优化」「实际案例」，确保深度与实用性。
